# Images

Drop the team photos here with these exact filenames so the
**"Life at CodeOrigin.Ai"** gallery on the website displays them:

| Filename            | Used for                                              |
|---------------------|-------------------------------------------------------|
| `team-photo-1.jpg`  | Engineering team in front of the office logo wall     |
| `team-photo-2.jpg`  | The full CodeOrigin.Ai team in front of the signage   |

Notes:
- Recommended size: ~1280px wide, landscape (4:3 works best).
- `.jpg` is expected. If you use `.png` or `.webp`, update the `src`
  attributes in `index.html` (search for `team-photo-1`).
- Until the files are added, the gallery shows a styled placeholder
  instead of a broken image.
